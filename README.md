# Portfolio Optimization - Time Series Forecasting

A time series forecasting project for GMF Investments to optimize portfolio management using TSLA, BND, and SPY.

## Assets
- TSLA: Tesla (high risk, high return)
- BND: Bond ETF (stable, low risk)
- SPY: S&P 500 ETF (diversified, moderate risk)

## Tasks
- Task 1: Data preprocessing and EDA
- Task 2: ARIMA and LSTM forecasting models
- Task 3: Future market trend forecasting
- Task 4: Portfolio optimization using Modern Portfolio Theory

## Setup
pip install -r requirements.txt
